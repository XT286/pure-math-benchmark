# Generated Candidates

This directory contains machine-generated but reference-grounded candidate query
packages.

These files are not part of the canonical reviewed benchmark source tree in
`benchmark/queries/`.

Each candidate package is organized by official MSC path and contains:

- `query.yaml`
- `rubric.yaml`
- `solution.yaml`
- `provenance.yaml`
- empty `reviews/area/` and `reviews/rubric/` directories for future expert review

The directory is regenerated by `npm run candidates:generate`.
