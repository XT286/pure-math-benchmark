# Public Benchmark Files

## v0.1 status

The current public corpus is a `seed release`, not a full benchmark release.

It is meant to demonstrate:

- taxonomy coverage across all subject families
- item schema usage
- research-style prompt design
- review metadata structure

## Included files

- `public_seed.jsonl`: representative seed items spanning all current subject families
- `../solutions/public_seed_solutions.jsonl`: canonical solution outlines for the first reviewed tranche
- `../reviews/public_seed_reviews.jsonl`: structured review records for the first reviewed tranche

## Current seed size

The current seed release contains `24` public items:

- an initial reviewed tranche of `12` cross-subject items with solution and review records
- an additional tranche of `12` representative expansion items broadening subfield coverage

## Scaling targets

The intended path from seed release to a fuller public benchmark is:

- `12` subject families
- about `4` subfields per family
- `2-3` finer topics per subfield
- `2-3` items per finer topic in the first substantial release

This points toward a future reviewed public corpus on the order of `360-480` items, with later expansion possible after editorial growth.

## Scope note

The seed release does not claim to be complete. It is a scaffold for expert review, contributor onboarding, and iteration toward a stronger public standard.
