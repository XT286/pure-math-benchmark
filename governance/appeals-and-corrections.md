# Appeals And Corrections

## Appeals

Contributors may appeal editorial decisions when they believe:

- an item was misunderstood
- a review misapplied the rubric
- a conflict of interest was overlooked

Appeals should be routed to the handling editor first, then to the Steering Committee if needed.

## Corrections

Corrections should be issued when:

- an item has ambiguous notation
- a canonical solution has a substantive gap
- metadata is materially wrong
- contamination or leakage risk changes

## Public record

Substantive corrections should appear in release notes or a public correction log.

## Retraction

Retraction is appropriate when an item is mathematically unsound, improperly sourced, or too compromised for continued use.
